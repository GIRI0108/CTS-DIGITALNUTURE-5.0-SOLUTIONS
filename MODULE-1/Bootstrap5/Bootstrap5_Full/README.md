Bootstrap5_Full Folder Structure

This structure follows the Bootstrap 5 exercise list.

Each task contains separate exercise folders.
Each exercise folder contains only:
index.html

Bootstrap local files should stay outside this folder like:
Bootstrap5/css/bootstrap.min.css
Bootstrap5/js/bootstrap.bundle.min.js

Because each index.html is inside:
Bootstrap5/Bootstrap5_Full/Task-xx/Exercise-xx/

The local Bootstrap path should be:
../../../css/bootstrap.min.css
../../../js/bootstrap.bundle.min.js

Code will be added later task by task.
